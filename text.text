สร้าง skill ให้ AI จดบันทึกทุกอย่างที่ทำไว้ใน .agents/sessions/ เพื่อเป็นการแชร์ context ทิ้งไว้ใน .md พอเวลาเปลี่ยนไปใช้ตัวอื่น จะได้เห็น context ก่อนหน้านี้
.
ข้างใน .agents/ จะมี:
📄 AGENTS.md — กติกาโปรเจคที่ AI อ่านก่อนทำงาน
📄 active.md — งานที่กำลังทำ ติดอะไร step ถัดไปคืออะไร
📁 sessions/ — checkpoint ของแต่ละรอบ เอาไว้ resume ต่อ
📁 topics/ — โน้ตยาวๆ ข้าม task
📁 private/ — โน้ตส่วนตัว auto-gitignore ไม่หลุดขึ้น repo
📄 index/repo-tree.md — แผนที่โปรเจค gen อัตโนมัติให้ AI อ่าน
📄 design.md
.
ดีไซน์มันรันครั้งเดียวจบ ยังแถม auto-detect ให้ว่าโปรเจคเป็น node / python / go / rust / docker แล้วบันทึกลง active.md ให้เอง รันซ้ำก็ไม่ทับของเดิม อยากลบก็ --clean จบ

สรุปหลักการทำงาน (TL;DR)

เริ่ม session:
1. อ่าน .agents/AGENTS.md
2. อ่าน .agents/active.md
3. สรุปกติกาสำคัญ + สถานะล่าสุด + next steps ก่อนลงมือ

ระหว่างทำงาน:
- ไม่ต้อง log อะไร แค่ทำงาน

จบ session:
1. อัปเดต .agents/active.md
   - Current Task
   - Status
   - What's Done
   - Blockers
   - Next Steps
   - Last Updated
   - Last Agent
2. อัปเดต .agents/AGENTS.md เฉพาะส่วนที่เป็นกติกาถาวรหรือ Last Agent ถ้าจำเป็น
3. สร้างไฟล์ checkpoint ใหม่ใน .agents/sessions/YYYY-MM-DD-{topic}.md
4. สรุปไฟล์ที่แก้ + สิ่งที่ต้องทำต่อ


เริ่ม session: อ่าน .agents/AGENTS.md และ .agents/active.md แล้วสรุปสถานะล่าสุด + next steps ก่อนลงมือ

จบ session: อัปเดต .agents/active.md, อัปเดต .agents/AGENTS.md ถ้ามีกติกาถาวรเปลี่ยน, สร้าง .agents/sessions/YYYY-MM-DD-{topic}.md แล้วสรุปสิ่งที่ทำ/ค้าง/ต่อไป
